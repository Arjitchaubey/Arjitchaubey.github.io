28nm HKMG NMOS Characterization & PDK Calibration

Project Overview

Developed a physics-based model of a 28nm HKMG NMOS transistor, implementing Density Gradient Quantization and Trap-Assisted Tunneling models. Performed DC calibration against a reference PDK model to achieve >95% agreement in subthreshold and saturation regions.

Core Accomplishments

  • Calibration: Tuned Philips Unified Mobility (PhuMob) and saturation velocity parameters to achieve >95% agreement.
  • AC Analysis: Executed Small-Signal Analysis (1 MHz) to extract Gate Capacitance ($C_{gg}$) curves across Accumulation, Depletion, and Inversion.

5-Stage Pipelined RISC-V Processor

Project Overview

Architected and implemented a 32-bit, 5-stage pipelined RISC-V (RV32I) processor from the ground up in Verilog. Supports R, I, S, U, and J-type instructions.

Performance Metrics

  • Frequency: Achieved timing closure with a WNS of 4.472 ns, indicating a theoretical max frequency of ~180 MHz.
  • Power: Analyzed post-implementation metrics, optimizing for 142 mW total power.
  • Resources: Achieved efficient utilization of ~1.5k Flip-Flops.

Challenges & Solutions

  • Challenge: Initial hazard control unit failed to handle control hazards from jumps.
    Solution: Re-architected control logic to stall the pipeline for two clock cycles upon detecting jump/branch instructions.
